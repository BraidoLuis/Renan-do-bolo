# Doce Gestão

Projeto demonstrativo de um SaaS para confeitaria, desenvolvido com React,
TypeScript, Vinext/Vite e CSS.

## Requisitos

- Node.js 20 ou superior
- npm
- Visual Studio Code (opcional)

## Como executar

1. Extraia o arquivo ZIP.
2. Abra a pasta `doce-gestao` no Visual Studio Code.
3. Abra o terminal integrado.
4. Instale as dependências:

   ```bash
   npm install
   ```

5. Inicie o projeto:

   ```bash
   npm run dev
   ```

6. Abra o endereço indicado no terminal.

## Acesso demonstrativo

Na tela inicial, escolha Administrador ou Cliente. Os campos já aparecem
preenchidos e a senha demonstrativa é `123456`.

## Observação

O projeto ainda não possui persistência em banco de dados. Produtos, contas,
pedidos, carrinho, orçamentos, estoque e notificações funcionam como protótipo
e voltam aos valores iniciais quando a página é recarregada.
